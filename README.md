## 🔧 Installation & Setup

### Option 1: Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Uzair3333/My-Portfolio.git
   cd My-Portfolio
   ```

2. Open in your browser:
   - Simply open `index.html` in your browser
   - Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

### Option 2: GitHub Pages
1. Push the repository to GitHub
2. Go to Settings → Pages
3. Select `main` branch as the source
4. Your portfolio will be live at `https://yourusername.github.io/My-Portfolio`

## 🎨 Customization

To customize this portfolio:

1. **Update Personal Info:** Edit the name and title in `index.html`
2. **Add Projects:** Modify `projects.html` to showcase your work
3. **Update Skills:** Edit `skills.html` with your technical expertise
4. **Change Colors:** Modify CSS variables in `assets/css/style.css`
5. **Add Images:** Replace placeholder images in `assets/images/`
6. **Social Links:** Update social media URLs in footer sections
